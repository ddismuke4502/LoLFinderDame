About LoLFinder

LoLFinder uses Riot's public Data Dragon service to display champion information such as roles, difficulty, and base stats. This project was built to demonstrate API consumption, search, sorting, filtering, and responsive UI design in vanilla HTML/CSS/JavaScript.

Fast discovery

Search champions by name with debounced input for smoother UX.

Filters that matter

Sort, filter by role, and cap difficulty to find your next main.

Responsive by default

Grid layout, mobile nav, and polished UI states across breakpoints.

Built with

HTML CSS JavaScript CSS Grid Data Dragon API

Project goals

  • API fetch + render pipeline
  • Search + sort + filters
  • Loading & empty states
  • Responsive layout system
